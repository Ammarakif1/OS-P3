

## Requirements
- Python 3.10+ (tested on Windows 10 PowerShell, but any OS with Python should work)

to run the command
write: 


python project3.py + the command 



| Command  | Usage / Arguments                                  | Description |
|----------|----------------------------------------------------|-------------|
| `create` | `python project3.py create index.idx`              | Creates a brand-new index 
| `insert` | `python project3.py insert index.idx key value`    | Inserts or updates a key/
| `search` | `python project3.py search index.idx key`          | Looks up a key and prints 
| `print`  | `python project3.py print index.idx`               | Prints all key/value pairs in 
| `load`   | `python project3.py load index.idx input.csv`      | Reads a CSV file 
| `extract`| `python project3.py extract index.idx output.csv`  | Writes every key/value pair
